# Face Detection and Counting using Dlib & OpenCV

A simple Face Detection project built with Python, OpenCV, and Dlib.

The application allows users to upload an image, automatically detect human faces, count the number of faces present, draw bounding boxes around each face, and save the processed image.

## Features

- Upload an image
- Detect faces automatically
- Count total faces in the image
- Draw bounding boxes around detected faces
- Label each detected face
- Save the processed image
- Display results using Matplotlib

## Technologies Used

- Python
- OpenCV
- Dlib
- Matplotlib
- Google Colab

## Installation

```bash
pip install opencv-python-headless dlib matplotlib numpy
```

## Usage

1. Open the notebook in Google Colab.
2. Run all cells.
3. Upload an image containing faces.
4. The program will:
   - Detect faces
   - Count faces
   - Draw bounding boxes
   - Save the output image


## Output Example

Input Image:
- Group photo containing multiple people

Output Image:
- Green bounding boxes around faces
- Labels for each face
- Total face count displayed
